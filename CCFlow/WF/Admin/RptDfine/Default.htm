フローレポートの定義説明
  • 一つのフローには三つのレポートフォームがあります。それぞれは、私が開始したフロー、当部門が開始したフロー、詳細検索です。
  • 各レポートは対応しています。検索コンポーネント、分析コンポーネント、詳細検索コンポーネント、対比分析コンポーネント、クロスレポートコンポーネントです。
  • 各レポートには、自分のコントロール権限、照会条件、表示内容のカスタマイズがあります。
  • 左をクリックして、レポートごとに権限を定義します。
私が始めたフロー
  • このレポートフォームは私の開始のフローだけを表示します。
  • レポート定義ツールで検索方式、表示内容を定義できます。
  • デフォルトではすべての人に権限があります。
私が参加したフロー
  • このレポートフォームは私の開始のフローだけを表示します。
  • レポート定義ツールで検索方式、表示内容を定義できます。
  • デフォルトではすべての人に権限があります。
うちの部署からのフロー
  • このレポートには、当部門によって開始されたフローのみが表示されます。
  • レポート定義ツールで検索方式、表示内容を定義できます。
  • 権限を定義
すべての部門から開始された
  • すべての部門が開始するフローを調べることができます。
  • レポート定義ツールで検索方式、表示内容を定義できます。
  • 権限を定義
